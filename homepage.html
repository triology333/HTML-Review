Descriptions

XML (Extensible Markup Language) does not specifically and actively do anything but is a method of simplification. A few examples of the simplifications XML can achieve is data sharing, data transport, data availability and changing platform. XML can be available in many kinds of reading machines (news feeds, voice machines, etc) and will store the data in plain text format.

<note>
   <date> 2015-09-01</date>
   <body> Don't to Take out the trash!</body>
</note>

DTD (Document Type Definition) is the structure and legal elemts/attributes of an XML document. DTD can only be declared within a XML file and wrapped within the !DOCTYPE tag definition.

<!DOCTYPE note [
<!ELEMENT note (to,from,heading,body)>
<!ELEMENT to (#PCDATA)>
]>

XSLT (Extensible Markup Language Transformations) is a language created for the purpose of transforming XML documents into other XML documents.

<tr >
<td > <xsl:value-of select="title"/ ></td>
<td > <xsl:value-of select="artist"/ ></td>
</tr >

HTML(HyperText Markup Language) is the most common standard markup language for Web pages. This allows any person to create their own websites with the most common tags “< b>, < p>, <body> < i>,etc”.

<body>
   <h1>This is a Heading</h1>
   <p>This is a paragraph. </p>
</body>

CSS(Cascading Style Sheets) which is used for the descriptions and “style” for HTML files. The CSS influences the HTML elements display on the screen and have the capability to be stored in CSS files or HTML files as a <style> tag.

<style>
p.center {
 text-align: center;
 color: red;
}
</style>

Web Formsare pages that your users request using their browser and can be written using a combination of HTML, server code, client-script and server controls. Many IDE’s give permission for users to drag and drop server controls onto the platform of Web Form pages.


JavaScript(JS) is one of many interpreted programming languages created for the purpose of software development. It conforms to the ECMAScript specifications and is classified as a high-leveled language with a compiler process instead of interpreter.

<script>
  var x = 5;
<script>

DOM(Document Object Model) has the responsibility of connecting web pages to scripts or programming languages by representing a certain structure. The DOM usually takes the form of representing a document with “logical trees” with each branch ending in a node (object).

<script type="text/javascript">
  var text = document.getElementById("one").innerHTML;
  alert("The first heading is " + text);
</script>

DHTML(Dynamic HyperText Markup Language) is a collection of multiple web-supporting technologies that create animated websites. There are many components that are factor ined into this static markup language: HTML, JavaScript, CSS, DOM,etc).


PHP(Hypertext Preprocessor) is a general purposed scripting tool. This preprocessor is mainly used in aspects of web development and html embedding.

<?php   echo "Hi, I'm a PHP script!"; ?>

ASP(application service provider) is a business providing computer-based services to customers over a network; such as access to a particular software application (such as customer relationship management) using a standard protocol (HTTP).


JSP(Java Server Pages) is technology that is able to provide a simplified way to create dynamic web content at a quick pass. And is responsible as a server-side technology for developing dynamic web pages. This is mainly used for implementing presentation layers of software development specifically HTML.

<% num1 = num1+num2 %>

Node.JSis an open-source, cross-platform, on the basis of a JavaScript runtime environment. It executes JavaScript code outside of a browser and correspond it with the works of the user.

Bootstrapis an a custom-made open source toolkit for that is compatible with the formats of HTML, CSS, and JS. Any work can be responsive to the bootstrap grid system, extensive prebuilt components, and powerful plugins built on jQuery.

HTTP(Hypertext Transfer Protocol) is a protocol structured by layer that is responsible for transmitting hypermedia documents, such as HTML. Hypertext Transfer Protocol Secure (HTTPS) is the first extension of the Hypertext Transfer Protocol (HTTP).and is used for secure communication over a network of a computer.


UDP(User Datagram Protocol) is an alternative communications protocol to Transmission Control Protocol (TCP) used primarily for establishing low-latency and less tolerating connections between applications on the internet. However, UDP is highly unreliable and connectionless protocol.
