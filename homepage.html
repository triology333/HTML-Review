Skeletal Tags

Skeletal Tags are tags that make the the structure of the HTML file. They are most important part because without them there would be no HTML Document.

Tag Name Purpose
<html> The html tag is what makes a HTML Document. It is the basis of the document itself.
<head> The head tag is a tag defining a specific area. The top of the page, it contains like links to CSS files and the title.
<title> The title tag is the name of the tab that you see at the top of the page. It gives the page a better name instead of the default which is just the name of the html file.
<body> The body tag is where all the content that you see on the actual website goes. This where you will find all your Text Tags and others go.
Text Tags

Text tag is a tag that handles the textual information for a HTML Document. It allows for easy organization on the text.

Tag Name Purpose
<p> The p tag stands for paragraph. It s a text tag that allows you write in paragraphs for your HTML document.
<h1> through <h6> The Tags h1 through h6 are all headers. the headers are defined by importance h1 being the most important and h6 being the least important. They are found in the body tag.
<blockquote> The blockquote tag is used to site information from other sources. They are ususally automatically indented by browswers.
<address> The address tag is used to define the contact inforamtion of the author/owner of the document. It is used either in a <body> tag or an <article> tag. Either way it represents the contact information for the document or the article.
<pre> The pre tag takes the the exact font and spaces of the html file and it will be used in the website.
Lists

Tag Name Purpose
<ol> The ol tag stands for ordered list. It makes ordered lists starting with the number 1
<ul> The ul tag stands for unordered list. It makes lists using bullet points.
<li> The li tag, is a nested tag. It is used in either the ol tag or the ul tag. It is a list element. It is its own tag becaus eit makes it easy for customization.
<dl> The dl tag is used to make description lists. It is used with dt and dd tags
<dt> The dt tag is the names that are to be described. It is a nested tag within the dl tag.
<dd> The dd tag is used to the actual description of the dt tag. It holds what is used to describe the name.
Structural Elements

Tag Name Purpose
<header> The header tag is used to hold important introductory information or even navigational information. It cannot be placed within another header element.
<footer> The footer tag is used to host information at the bottom of the page. It ususallycontains, copyright information. sitemap, back to top links, etc. It is possible to have several footer elements in one document.
<nav> The nav tag is used to create a set of navigation links. It only used for the major block of navigation links. Ie: the ones you would see constantly on a webpage.
Links and Media Tags

Tag Name Purpose
<a> The html tag is what makes a HTML Document. It is the basis of the document itself.
<link> The head tag is a tag defining a specific area. The top of the page, it contains like links to CSS files and the title.
<img> The img tag defines an image in an HTML document. It has two attibutes: src and alt. Src which holds the image file name it usually ends in jpg,bu it could also be a hyper link to and image that was found on the web.
<figure> The figure tag is for self contained content. it is independent to the structure of the document.
<figcaption> The figcaption tag defines the caption for a figure element. The child placement of this tag does not matter.
<video> The video tag specifies video content, such as a movie clip of video streams.
<audio> The audio tag is used to define sound in a html document, such as music or other audio streams.
<object> The object tag defines an embedded object within an HTML document. This element is used to embed multimedia.
<param> this tag is used to define parameters fo the object tag.
<embed> The embed tag defines a container for external applications or interactive content.
Tables

Tag Name Purpose
<table> The table tag is used to define an HTML table. It consists of tr, th, and td elements.
<tr> The tr tag is used to define a row in a HTML table.
<th> The th tag is used to define the header for each column in the HTML table.
<td> The td tag is used to define the table data in a HTML table. It is usually nested in the tr tag.
<caption> The caption tag is used to define a table caption. it must be used immediately after the table tag.
<thead> The thead tag is used to group header content in an HTML table.
<tbody> The tbody tag is used to group the body content of the HTMK table.
<colgroup> The colgroup tag is used group more than one HTML table columns for formatting.
<col> The col tag represents the column properties within a colgroup element
<tfoot> The tfoot tag is used to group footer content in an HTML table. It isued with the thead and tbody elements.
