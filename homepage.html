Skeletal Tags

Skeletal Tags are tags that make the the structure of the HTML file. They are most important part because without them there would be no HTML Document.

Tag Name Purpose Example
<html> The html tag is what makes a HTML Document. It is the basis of the document itself. <html>
</html>
<head> The head tag is a tag defining a specific area. The top of the page, it contains like links to CSS files and the title. <html>
<head>
<title>Hello World!<title>
</head> </html>
<title> The title tag is the name of the tab that you see at the top of the page. It gives the page a better name instead of the default which is just the name of the html file. Austria
<body> The body tag is where all the content that you see on the actual website goes. This where you will find all your Text Tags and others go. <html>
<head>
<title>Hello World!<title>
</head> <body> <h1>Hello World!&</h1> </body> </html>
Text Tags

Text tag is a tag that handles the textual information for a HTML Document. It allows for easy organization on the text.

Tag Name Purpose Example
<p> The p tag stands for paragraph. It s a text tag that allows you write in paragraphs for your HTML document. <p>
This is a start of a really good paragraph. <p>
<h1> through <h6> The Tags h1 through h6 are all headers. the headers are defined by importance h1 being the most important and h6 being the least important. They are found in the body tag. <html>
<h1> This is a heading</h1>
<h2> This is a heading </h2>
<h3> This is a heading</h3>
<h4> This is a heading</h4>
<h5> This is a heading</h5>
<h6> This is a heading</h6>
</html>
<blockquote> The blockquote tag is used to site information from other sources. They are ususally automatically indented by browswers.
<blockquote cite ="https://www.w3schools.com/tags/tag_blockquote.asp"> The <blockquote> tag specifies a section that is quoted from another source.</blockquote>
<address> The address tag is used to define the contact inforamtion of the author/owner of the document. It is used either in a <body> tag or an <article> tag. Either way it represents the contact information for the document or the article. Austria
